About the project

The KuopioHema project is a collection of helper apps intended for use in pediatrics, especially pediatric haematology and oncology. The app creator does not guarantee or take responsibility for their function. All apps are licensed under the GNU GPLv3 license.

Browser support:

Apps support all common modern browsers (Edge, Firefox, Chrome, Safari). Internet Explorer and old versions of other browsers are not supported.

Data security:

All apps on this site run entirely on the user's machine, using the browser's JavaScript engine. No user-entered data is transmitted to the server.
